# Flask Test API-MVC App

### Python Env

Create python env

```
python3 -m venv env
```

Run Windows

```
env\Scripts\activate
```

Run on Unix-MacOS

```
source env/bin/activate
```

### Requierements.txt file

Store installs packages

```
pip freeze > requirements.txt
```

Install requirements file

```
pip install -r requirements.txt
```
